## How to Use These Templates

**Design Tools:**
- Canva (easiest - has LinkedIn carousel templates)
- Figma (more customizable)
- Adobe Express

**Optimal Format:**
- Size: 1080px √ó 1080px per slide
- File type: PDF (LinkedIn's preferred format)
- Slides: 6-10 slides maximum
- Font: Sans-serif, high contrast

**Design Tips:**
1. Use consistent branding (colors, fonts)
2. One key point per slide
3. Large, readable text (min 40pt)
4. High contrast (dark text on light background)
5. Include your logo/website on last slide
6. Number slides (e.g., "2/8" in corner)

## DESIGN GUIDELINES FOR ALL CAROUSELS

### Color Palette Recommendations
**Option 1: Professional**
- Primary: #1E40AF (Deep Blue)
- Secondary: #10B981 (Green)
- Accent: #F59E0B (Amber)
- Background: #F9FAFB (Light Gray)

**Option 2: Modern Tech**
- Primary: #6366F1 (Indigo)
- Secondary: #EC4899 (Pink)
- Accent: #14B8A6 (Teal)
- Background: #FFFFFF (White)

**Option 3: Bold**
- Primary: #7C3AED (Purple)
- Secondary: #EF4444 (Red)
- Accent: #3B82F6 (Blue)
- Background: #F3F4F6 (Gray)

### Typography
- Headers: Bold sans-serif (Montserrat, Poppins, Inter)
- Body: Clean sans-serif (Inter, Roboto, Open Sans)
- Minimum size: 40pt for readability on mobile

### Layout Best Practices
1. 40% white space minimum
2. Max 3 colors per slide
3. Consistent header placement
4. Number slides (e.g., "3/8" in top corner)
5. Logo/watermark on every slide (bottom corner)
6. High contrast text (WCAG AA minimum)

### Call-to-Action Slides
- Use button-style CTA
- Include QR code when possible
- Repeat your URL clearly
- Add urgency ("Start free", "Try now")

## POSTING BEST PRACTICES

1. **Post carousel as PDF** (not images) - better reach
2. **First comment:** Add link to tool immediately
3. **Tag relevant people** if you reference them
4. **Use 3-5 hashtags:** #CareerAdvice #BusinessNiche #Entrepreneurship
5. **Pin top comment:** Your tool link + value statement
6. **Engage immediately:** Reply to comments within first hour
